# Make the site live (one-time, ~30 seconds)

Open: https://github.com/jonray757-hue/bscg-fact-finder/settings/pages

1. **Build and deployment** → **Source** → **Deploy from a branch**
2. **Branch** → `main` → folder **`/ (root)`**
3. Click **Save**

Live in 1–2 minutes:

https://jonray757-hue.github.io/bscg-fact-finder/
